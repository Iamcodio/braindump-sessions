# Brain Dump Quick Start

**Copy-paste this into new Claude chat to begin session:**

---

It's [CHECK TIME] and I'm ready for a brain dump.

**Baseline Check:**
- Sleep: [hours + quality]
- Exercise: [yes/no + sunlight]
- SUDS: [0-10]
- Energy: [1-5]
- Outlook: [1-5]

Let's go.

---

**CHAT TITLE SUGGESTION:** Brain Dump 2026-01-07
(Claude will provide this automatically)

---

## End of Day

When done for the day, just say:

```
Run end of day
```

Claude will:
1. Review entire chat
2. Extract metrics + insights
3. Create archive artifact
4. Save to: `~/09-personal/BrainDumpSessions/sessions/YYYY-MM/YYYY-MM-DD_brain_dump_archive.md`

---

## Monthly Report

To generate monthly summary:

```
Generate monthly report for [month year]
```

Example: "Generate monthly report for December 2025"

---

## SUDS Reference

**0-1:** Calm, at peace  
**2-3:** Light background anxiety  
**4-5:** Notable anxiety, restless  
**6-7:** High anxiety, fight/flight  
**8-9:** Severe distress, near panic  
**10:** Highest distress ever experienced
